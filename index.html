Welcome to Rexel

A Simple, Powerful Programming Language Based on Lua

What is Rexel?

Rexel is a minimalistic programming language designed to be simple, powerful, and intuitive, built on the foundation of Lua. It is designed for developers who want a language that balances simplicity with flexibility, without unnecessary complexity.

Why Use Rexel?

Rexel allows you to create clean, readable code while retaining powerful functionality. With its focus on simplicity, it helps you get started quickly and easily without needing to worry about complex syntax.

  • Simple Syntax - Easy-to-read code with minimal boilerplate.
  • Powerful - Supports key features for modern programming.
  • Flexible - Customize the language for your needs.

Installation

To get started with Rexel, you will need to download the interpreter and run your programs locally.

  1. Step 1: Download the Rexel interpreter from the GitHub repository.
  2. Step 2: Install the interpreter by following the instructions in the README file.
  3. Step 3: Write your first program in Rexel and run it with the interpreter.

Example Code

Here's an example of what Rexel code looks like:


** This is a comment
set x 10
set y 20

function greet(name) {
    set message "Hello, " .. name
    print(message)
}

greet("Rexel")
